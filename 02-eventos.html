JQuery - Eventos

O que são eventos?

Os eventos são o momento em que certa ação deve ser executada. Por exemplo, podemos chamar um script quando:

  • Quando mover o mouse por um elemento.
  • Quando clicar em um elemento.
  • Quando selecionar uma opção em um input do tipo radio.
Segue alguns eventos DOM:

Eventos do Mouse Eventos de Teclado Eventos de Formulário Eventos de Documento/janela
click keypress submit load
dblclick keydown change resize
mouseenter keyup focus scroll
mouseleave blur unload

Exemplo:

$("p").click(function (){
	//Acontece algo quando clicar em qualquer parágrafo
});

Eventos do JQuery usados comumente

$(document).ready()

O método $(document).ready() executa algo aconteça quando a página for carregada.

click()

O método click() executa algo aconteça quando o elemento for clicado. Exemplo: 

$("#botao").click(function(){//Acontece algo});

dblclick()

O método dblclick() executa algo quando o usuário clicar duas vezes (double-click) em um elemento.

mouseenter()

O método mouseenter() executa algo quando o usuário passar o mouse por cima de um elemento.

mouseleave()

O método mouseleave() executa algo quando o usuário tirar o mouse do elemento.

mousedown()

O método mousedown() executa algo quando o usuário apertar o botão do mouse enquanto estiver com o cursor sob o elemento.

mouseup()

O método mouseup() executa algo quando o usuário soltar o botão do mouse enquanto estiver com o cursor sob o elemento.

hover()

O método hover() pega dois parâmetros, uma função para ser executada quando o cursor entrar no elemento e outra quando sair.
Exemplo:

$("#elemento").hover(function(){
	//Primeira função
}, function(){
	//Segunda função
}
);
focus()

O método focus() executa algo quando foca (seleciona) um campo do formulário.

blur()

Inversamente do focus, este executa uma função quando o campo perder o foco.

on()

O método on() executa uma função quando acontecer um ou mais eventos. 


$("p").on("click", function(){
	//Executar esta função quando clicar em algum parágrafo.
});

$("p").on({
	mouseenter: function(){
		//ação quando o mouse entrar no elemento
	},
	mouseleave: function(){
		//Ação quando o mouse sair do elemento
	},
	click: function(){
		//Ação quando o elemento for clicado
	}
});
