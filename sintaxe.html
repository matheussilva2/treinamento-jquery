JQuery - Sintaxe

A sintaxe básica do JQuery é $(seletor).acao();

  • $ é o caractere para chamar o JQuery
  • O (seletor) é usado para achar elementos html
  • A acao() é a ação JQuery que atuará no elemento

Exemplos:
$(this).hide() - Esconde o elemento atual
$("p").hide() - Esconde todos os elementos <p> da página;
$(".test").hide() - Esconde todos os elementos da class test
$("#test").hide() - Esconde todos os elementos do id test
