jQuery - Stop

O método Stop()

Observação

Este conteúdo têm como pré-requisito o método Animate().
Clique aqui para acessar a página sobre o método Animate().

O método Stop serve para interromper uma animação.



Target
Código 
$(document).ready(function(){
	$("#ex1start").click(function(){
		$("#ex1target").animate({
			height: '200',
			width: '200'},
			3000);
	});
	$("#ex1stop").click(function(){
		$("#ex1target").stop();
	});
});
