JQuery - Seletores

Os seletores do JQuery são utilizados para manipular elementos HTML. Como citado nesta pagina, sua estrutura é $("elemento").acao(). Todos os seletores JQuery começam com $().
Por exemplo, o seletor $("p") seleciona todos os parágrafos da página.

Exemplo


$(document).ready(function(){
	$("button").click(function{
		$("p").hide(); //Esconde todos os parágrafos da página se algum botão for clicado.
	});
});

Selecionando elementos por ID

A seleção de elementos html através do seu id é feita por $("#id")

Exemplo


$(document).ready(function(){
	$("button").click(function(){
		$("#test").hide(); //Esconde o elemento do id #test quando algum botão for clicado
	});
});

Selecionando elementos por class

Para selecionar por class, usamos $(".teste").

Outros seletores

Sintaxe Descrição
$("*") Selecionar todos os elementos
$(this) Selecionar elemento atual
$("p.intro") Selecionar todos os parágrafos da class intro
$("p:first") Selecionar o primeiro parágrafo
$("ul li:first") Selecionar o primeiro parágrafo
$("ul li:first-child") Seleciona o primeiro li de cada ul
$("[href]") Seleciona todos os elementos com o atributo href
$("a[target='_blank']" Seleciona todos os elementos a que o atributo target seja igual a _blank
$("a[target!='_blank']") Seleciona todos os elementos a que o atributo target seja diferente de _blank
$(":button") Seleciona todos os elementos button e input que seja do tipo button
$("tr:even") Seleciona todos os elementos tr que sejam pares
$("tr:odd") Seleciona todos os elementos tr que sejam ímpares
