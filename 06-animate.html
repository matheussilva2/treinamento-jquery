jQuery - Animate

A estrutura do animate é:

$(seletor).animate({parametros}, velocidade, callback);
  • Os parâmetros são as propriedades CSS alteradas na animação.
  • Como outros métodos animados, a velocidade é: SLOW, FAST ou em milisegundos
  • Callback é uma função que será executada quando a animação terminar.
Target

Código 
$("#exemplo1btn").click(function(){
	$("#exemplo1").animate({left:'250px'});
});

Manipulando mútiplas propriedades CSS

Separe as propriedades com vírgulas para fazer mútiplas animações:

Target

Código 
$("#exemplo2btn").click(function(){
	$("#exemplo2").animate({
		left: '250px',
		opacity: '0.5',
		height: '150px',
		width: '150px'
	});
});

É possível manipular todos os elementos CSS com o método animate()?

Sim, mas lembre-se que todos as propriedades devem ser escritas em camel-case, por exemplo: use paddingLeft em vez de padding-left ou marginRight em vez de margin-right.

Usando valores relativos

Esta forma de fazer a animação permite incrementar valores com os operadores aritméticos += e -=

Código 
$("#exemplo2btn").click(function(){
	$("#exemplo2").animate({
		left: '250px',
		opacity: '0.5',
		height: '+=150px',
		width: '-=150px'
	});
});

Usando valores pré-definidos

Você pode usar os valores "show", "hide" e "toggle":

Código 
$("#exemplo2btn").click(function(){
	$("#exemplo2").animate({
		height: 'toggle'
	});
});

Fila de animações

Você pode criar uma fila de animação, jQuery irá executar uma por uma.

Target

Código 
$("#exemplo3btn").click(function(){
	$("#exemplo3").animate({height:'300px',opacity:'0.4'},"slow");
	$("#exemplo3").animate({width:'300px',opacity:'0.8'},'slow');
	$("#exemplo3").animate({height:'100px',opacity:'0.4'});
	$("#exemplo3").animate({width:'100px',opacity:'0.8'},'slow');
});
